# Neural Networks

## Backpropagation
Backpropagation is the algorithm used to train neural networks. It calculates gradients by propagating error backwards through the network. This allows the network to update weights and learn from data.

## Activation Functions
Common activation functions include ReLU, sigmoid, and tanh. ReLU is fast and helps prevent vanishing gradients. Sigmoid outputs between 0 and 1, good for binary classification.

## Gradient Descent
Gradient descent optimizes network weights by moving in the direction of steepest descent. The learning rate controls how big each step is.
